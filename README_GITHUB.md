# Simulador final - Análisis e Interpretación de Datos

## Qué incluye
- 20 preguntas tipo examen final
- 4 opciones por pregunta
- Calificación automática
- Explicación breve para memorizar rápido
- Todo en un solo archivo `index.html`

## Cómo subirlo a GitHub Pages
1. Crea un repositorio nuevo en GitHub.
2. Sube el archivo `index.html` al repositorio.
3. Opcionalmente sube también este `README_GITHUB.md`.
4. Ve a **Settings > Pages**.
5. En **Build and deployment**, selecciona **Deploy from a branch**.
6. Elige la rama `main` y la carpeta `/root`.
7. Guarda los cambios.
8. GitHub te dará una URL pública para abrir la aplicación.

## Cómo probarlo localmente
Solo abre `index.html` en tu navegador.

## Personalización rápida
- Las preguntas están dentro de una constante llamada `questions` en el JavaScript.
- Puedes cambiar textos, respuestas correctas o agregar más reactivos.
